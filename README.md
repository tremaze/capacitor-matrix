# @tremaze/capacitor-matrix

A capacitor plugin wrapping native matrix SDKs

## Install

To use npm

```bash
npm install @tremaze/capacitor-matrix
````

To use yarn

```bash
yarn add @tremaze/capacitor-matrix
```

Sync native files

```bash
npx cap sync
```
